# Tendering
后端 SpringBoot + 前端 JQuery、ECharts 的采购中标可视化系统 

这是 2018 年做的一个项目<br>
后端使用 SpringBoot + Mybatis；<br>
前端使用 JQuery 进行请求和 ECharts 进行可视化，前端暂时用的 Jsp 页面
